Next.js 是基于 React 的框架，它提供了一个简单的构建工具，可以让你快速构建一个网站。

## 安装

> 前置条件：安装 Node.js、npm

```bash
npm init -y
npm install next react react-dom

# + "next": "^12.1.6",
# + "react": "^18.1.0",
# + "react-dom": "^18.1.0"
```

## 创建`pages`目录

```bash
mkdir pages && cd pages && touch index.js
```

```js
function HomePage() {
  return <div>Welcome to Next.js!</div>
}

export default HomePage
```

## 启动服务

```bash
npm run dev

#npm run dev --port=8888
#

#> next dev
#

```
