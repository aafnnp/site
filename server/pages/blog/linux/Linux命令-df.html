Linux命令df

October 11, 2020

Published a year ago

df命令用于显示目前在 Linux 系统上的文件系统磁盘使用情况统计。

使用方法

同样可以使用df folder显示文件夹的使用情况

df folder

参数

  1. -a, --all 包含所有的具有 0 Blocks 的文件系统
  2. --block-size={SIZE} 使用 {SIZE} 大小的 Blocks
  3. -h, --human-readable 使用人类可读的格式(预设值是不加这个选项的...)
  4. -H, --si 很像 -h, 但是用 1000 为单位而不是用 1024
  5. -i, --inodes 列出 inode 资讯，不列出已使用 block
  6. -k, --kilobytes 就像是 --block-size=1024
  7. -l, --local 限制列出的文件结构
  8. -m, --megabytes 就像 --block-size=1048576
  9. --no-sync 取得资讯前不 sync (预设值)
  10. -P, --portability 使用 POSIX 输出格式
  11. --sync 在取得资讯前 sync
  12. -t, --type=TYPE 限制列出文件系统的 TYPE
  13. -T, --print-type 显示文件系统的形式
  14. -x, --exclude-type=TYPE 限制列出文件系统不要显示 TYPE
  15. -v (忽略)
  16. --help 显示这个帮手并且离开
  17. --version 输出版本资讯并且离开
