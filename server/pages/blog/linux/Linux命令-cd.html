Linux命令cd

October 13, 2020

Published a year ago

cd命令用于切换当前工作目录。

~ 也表示为 home 目录 的意思， . 则是表示目前所在的目录， .. 则表示目前目录位置的上一层目录

使用方法

#切换到某个目录
cd foldername

#当前所在目录
cd .

#切换到上一层目录
cd ..

#切换到home目录
cd ~

#切换到根目录
cd /
