在 HTML 中，列表（Lists）是一种用于组织信息的结构。列表可以分为有序列表（Ordered Lists）和无序列表（Unordered Lists）两种类型。

无序列表：用于表示没有特定顺序或层次的项目列表。在 HTML 中，无序列表使用ul标签来定义，每个项目使用li标签来定义。示例如下：

```html
<ul>
  <li>项目 1</li>
  <li>项目 2</li>
  <li>项目 3</li>
</ul>
```

有序列表：用于表示按照特定顺序或层次的项目列表。在 HTML 中，有序列表使用ol标签来定义，每个项目使用li标签来定义。示例如下：

```html
<ol>
  <li>第一项</li>
  <li>第二项</li>
  <li>第三项</li>
</ol>
```

除了基本列表之外，HTML 还提供了其他类型的列表，如描述列表（Description Lists）和嵌套列表（Nested Lists）。

描述列表：用于表示一组术语及其定义的列表。在 HTML 中，描述列表使用dl标签来定义，每个术语使用dt标签来定义，每个定义使用dd标签来定义。示例如下：

````html
<dl>
  <dt>术语 1</dt>
  <dd>定义 1</dd>
  <dt>术语 2</dt>
  <dd>定义 2</dd>
  <dt>术语 3</dt>
  <dd>定义 3</dd>
</dl>
``` 嵌套列表：用于表示列表项目之间的嵌套关系。在 HTML
中，可以将一个列表放置在另一个列表的项目中，以创建嵌套列表。示例如下： `html
<ul>
  <li>项目 1</li>
  <li>
    项目 2
    <ul>
      <li>项目 2.1</li>
      <li>项目 2.2</li>
    </ul>
  </li>
  <li>项目 3</li>
</ul>
`
通过使用列表，可以有效地组织和呈现信息，使其更易于阅读和理解。在实际项目中，通常会使用
CSS 对列表进行样式设计，以使其更加美观和有用。
````
