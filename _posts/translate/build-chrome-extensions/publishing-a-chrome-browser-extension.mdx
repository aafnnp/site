[Chrome Store](https://chrome.google.com/webstore/devconsole)

向 chrome store 发布插件需要账户，并且一次性支付 5 刀

![ltzRTH](https://cdn.jsdelivr.net/gh/manonicu/pics@master/uPic/ltzRTH.jpg)

注册并支付后，点击做侧边栏选项卡，点击新建项目

![1nDx44](https://cdn.jsdelivr.net/gh/manonicu/pics@master/uPic/1nDx44.jpg)

提交`zip`文件后，需要填写一些必须的字段：

- 描述：关于此扩展做什么的
- 类别：扩展类型
- 语言：插件的主要语言
- 图标：商店图标（128\*128）
- 屏幕截图：提供至少一张 1280*800 或 640*400 的屏幕截图

在`隐私`选项上：

- 简短介绍
- 权限：使用了哪些权限
- 数据：需要使用哪些数据

如无法提交，点击以下链接查看可能的原因。
