1. 复制`favicon`文件到`public`目录下

![kbo0B5](https://cdn.jsdelivr.net/gh/manonicu/pics@master/uPic/kbo0B5.jpg)

2. 编辑`pages/_app.js`

```js
import Head from 'next/head'

function MyApp({Component, pageProps}) {
  return (
    <>
      {/* the favicon */}
      <Head>
        <link rel="shortcut icon" href="/favicon.ico" />
      </Head>
      {/* the favicon */}
      <Component {...pageProps} />
    </>
  )
}

export default MyApp
```
