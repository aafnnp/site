Boolean 有两个值 true/false。

| 数据类型  | 转换为 true 的值 | 转换为 false 的值 |
| --------- | ---------------- | ----------------- |
| Boolean   | true             | false             |
| String    | 任意飞空字符串   | 空字符串          |
| Number    | 任何非零值       | 0 和 NaN          |
| Object    | 任何对象         | null              |
| Undefined |                  | undefined         |
