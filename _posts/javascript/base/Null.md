Null 和 Undefined 一样，只有一个值，就是 null

null 表示空对象指针，使用 typeof null 会返回 object

null 与 undefined 的区别

```javascript
typeof null; //object
typeof undefined; //undefined

null == undefined;
null !== undefined;
```
