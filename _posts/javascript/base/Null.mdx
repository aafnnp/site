原始类型 null 表示有意不存在值，因此，通常将 null 复制给变量，以表示变量已被声明和初始化并且当前没有值，但未来会有值。

Null 和 Undefined 一样，只有一个值，就是 null

null 表示空对象指针，使用 typeof null 会返回 object

null 与 undefined 的区别

```js
typeof null //object
typeof undefined //undefined

null == undefined
null !== undefined
```
