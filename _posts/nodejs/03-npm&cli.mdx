## npm

`npm`即 `Node.js Package Manager`，为 node.js 提供在线软件包、模块存储库，提供了命令行作为 node.js 的版本管理和依赖管理。

![Nrqk6V](https://cdn.jsdelivr.net/gh/manonicu/pics@master/uPic/Nrqk6V.png)

使用 npm 安装模块

```shell
# npm install <module name>

npm install express
```

全局安装

```shell
# 使用 -g 或者 global全局安装模块
npm install express -g
```

卸载模块

```shell
#npm uninstall <module name>

npm uninstall express
```

搜索模块

```shell
# npm search <module name>

npm search express
```

显示当前安装模块

```shell
npm ls
```
