## 为什么要创建自己的配置？

通过一步步创建自己的配置可以学习到`webpack`和`babel`、`eslint`等这些工具的知识。

随着开发时间及项目的增长，需要集成更多的工具到项目中，比如：

- webpack loaders（less、sass）
- 服务器渲染
- ES next
- mobx、redux等
- jest、cypress等


## 初始化项目

``` shell
#创建并进入目录
mkdir react-config-tutorial && cd react-config-tutorial

#初始化npm项目
npm init -y

#安装react
npm install react react-dom
```
