ReactJS 是一个声明性、高效和灵活的 JavaScript 库，用于构建可重用的 UI 组件。这是一个开源的、基于组件的前端库，仅负责应用程序的视图层。它最初由 Facebook 开发和维护，后来用于 WhatsApp 和 Instagram 等产品。

## 前置技术

- babel
- webpack
- nodejs && npm
- html && css
