此外，`JSX`的属性需要不同的语法。

由于`JSX`实际上是 JavaScript，JavaScript 使用 camelcase 命名约定（即“camelcase”），因此属性的编写方式与 HTML 不同。

比如，`className`和`class`是不同的，`className`是 HTML 的属性，`class`是 JavaScript 的属性。

```jsx
<div className="container"><div
```
