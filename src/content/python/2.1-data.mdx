## 原始数据类型

- 整形：`int`
- 浮点型：`float`
- 字符串：`str`

## 无类型

```python
email_address = None
```

`None` 是一个特殊的值，表示空值。

```python
if email_address:
    print('email address provided.')
```

## 元组

元组是一种有序的数据集合，元组中的元素可以是不同的数据类型。

```python
s = ('a', 'b', 'c')
```

有时候`()`可以省略，但是不建议这样做。

```python
s = 'a', 'b', 'c'
```

通过索引访问元组中的元素，索引从 0 开始。

```python
s = ('GOOG', 100, 490.10, 'Google Inc.')
name = s[0] # GOOG
price = s[2] # 490.10
shares = s[1] # 100
```

无法修改元组中的元素。

```python
s[1] = 200
# TypeError: 'tuple' object does not support item assignment
```

元组可以被重新赋值，但是不能改变元组的长度。

```python
s = ('GOO', 200, 490.10, 'Google Inc.')
```

## 元组的包装和拆包

```python
s = ('GOOG', 100, 490.10, 'Google Inc.')
name, shares, price, company = s
```

## 元组和列表

元组通常由不同的数据，而列表是相同类型的数据队列。元组表示的是结构，而列表表示的是顺序

- 都是序列
- 都可以存储任何数据类型
- 可以通过索引访问

```python
record = ('GOOG', 100, 490.1)       # 一些不同的数据

symbols = [ 'GOOG', 'AAPL', 'IBM' ]  # 一些相同类型的数据
```

## 字典

字典是一种可变的数据结构，它是一种映射，也就是一种键值对的集合。

```python
s = {
    'name': 'GOOG',
    'shares': 100,
    'price': 490.1
}

s['name'] # GOOG
s['shares'] # 100
s['price'] # 490.1
```
