## 浏览器插件类型

通过浏览器插件可以实现一些自定义功能，比如：自定义背景、字体、浏览模式等。

在所有浏览器中，chrome 是最具扩展性，firefox、edge 等也不错。而且有些插件通用兼容 firefox 和 edge。

主题插件：[【All Chrome Themes】](https://chrome.google.com/webstore/category/themes?hl=en-GB)

新窗口插件：比如[daily.dev](https://daily.dev/) [【Some new tab extension on chrome】](https://chrome.google.com/webstore/category/collection/customize_your_new_tab_page)

弹窗插件：

- [Lighthouse](https://chrome.google.com/webstore/detail/lighthouse/blipmdconlkpinefehnmjammfjpmpbjk)
- [Developer toolbar](https://chrome.google.com/webstore/detail/web-developer/bfbameneiokkgbdmiekhjnmfkcnldhhm)
- [Metamask](https://chrome.google.com/webstore/detail/metamask/nkbihfbeogaeaoehlefnkodbefgpgknn)
