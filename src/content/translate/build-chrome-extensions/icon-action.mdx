## 浏览器扩展图标 - action

修改`public/manifest.json`

```json
{
  "action": {},
  "background": {
    "service_worker": "background.js"
  }
}
```

修改`public/background.js`

```js
chrome.action.onClicked.addListener(() => {
  chrome.tabs.create({url: './new-tab.html', active: true})
})
```

当点击图标，将会导航到新的选项卡
