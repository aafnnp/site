overflow 属性用于控制元素内容的溢出方式。当元素内容超出其指定的尺寸时，可以使用 overflow 属性来指定如何处理溢出内容。

overflow 属性有四个可能的值：

- visible（默认值）：内容超出元素尺寸时不进行剪切，可以继续显示在元素外面。这可能会导致元素之外的内容被覆盖。
- hidden：超出元素尺寸的内容将被剪切，不会被显示出来。这可以用于隐藏不想被用户看到的内容。
- scroll：超出元素尺寸的内容会被剪切，但会显示一个滚动条，用户可以使用滚动条来查看隐藏的内容。
- auto：与 scroll 值相似，超出元素尺寸的内容会被剪切，但只有在需要滚动时才会显示滚动条。

overflow 属性通常与 height 或 width 属性一起使用，以指定元素的尺寸和如何处理溢出内容。例如：

```css
div {
  width: 200px;
  height: 100px;
  overflow: scroll;
}
```

这个样式规则将创建一个宽度为 200 像素、高度为 100 像素的 div 元素，并使用 overflow: scroll 属性将超出这些尺寸的内容剪切并添加滚动条。
