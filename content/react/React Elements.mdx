React Elements 使用 `HTML` 标签来构建组件。

```html
<h1>My Header</h1>
<p>My paragraph> <button>My button</button></p>
```

我们使用一个名为`JSX`的特性编写 React Elements。

然而，因为`JSX`实际上只是 JavaScript 函数（而不是 HTML），所以语法有点不同。

与 HTML 不同，单标记元素（如 img 元素）必须是自动关闭的。它们必须以正斜杠结尾`/`：

```jsx
<img src="my-image.png" />
<br />
<hr />
```
