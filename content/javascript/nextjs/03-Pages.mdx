在 Next.js 中，页面是一个 React Component，从 pages 目录导出。每个页面都基于其文件名与一个路由关联。

比如：

- `pages/index.js` 与 / 映射
- `pages/about.js` 与 /about 映射
- `pages/blog/[slug].js` 与 /blog/[slug] 映射

创建 posts 目录和 first.js

```js first.js
import Link from 'next/link'
export default function First() {
  return (
    <>
      <h1>My First Post</h1>
      <h2>
        <Link href="/">
          <a>Home</a>
        </Link>
      </h2>
    </>
  )
}
```

```js index.js
import Link from 'next/link'

function HomePage() {
  return (
    <>
      <div>Welcome to Next.js!</div>
      <Link href="/posts/first">
        <a>First Post</a>
      </Link>
    </>
  )
}

export default HomePage
```
